### Configure

```bash
# Run
$ composer install

# generate laravel APP_KEY
$ php artisan key:generate

# install npm
$ npm install

# generate mixing
$ npm run dev

# starting server using artisan
$ php artisan serve
```

Configure **.env** , copy paste .env.example and remove **.example**
